openpilot Replay Clipper

Local WSL queue UI for route clips. Paste a full connect URL or a raw dongle|route id, queue jobs, and watch them move through the renderer.

Outputs land in shared/. Valid JWT tokens are cached only in the repo's ignored .cache/ folder.

JWT token

Use this only when a route is private. The token is stored in the repo's ignored .cache/ folder so it survives restarts without becoming committable.

Queue a clip

If you paste a raw dongle|route id and leave the timing fields at their defaults, the web UI expands that to the full route automatically. Full clip URLs from comma Connect keep using the timing embedded in the URL.
